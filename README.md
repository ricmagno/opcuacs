# OPCUACS

A high-performance, lightweight OPC UA Client and Server built with Rust.

## Requirements

1. **Fast and Light**: Built with Rust for optimal performance and resource efficiency.
2. **Cloud Native**: Designed to run in containers orchestrated by Kubernetes.
3. **UIX Explorer**: Web-based interface to manage client and server connections.
4. **Connectivity**:
    - **Client**: Connects to OPC UA servers (primarily PLCs).
    - **Server**: Serves tags to applications like SCADA, Matlab, and other utilities.
