# CONTRIBUTING.md

Welcome to the OPCUACS project. This document outlines the process for contributing to the codebase.

## For Human Contributors

- Follow the coding standards defined in [AGENTS.md](AGENTS.md).
- Ensure all changes are covered by tests.
- Provide clear commit messages.

## For AI Agents

- Always work within a task boundary.
- Create and follow implementation plans for complex changes.
- Update `MEMORY.md` after completing a task.
- Ensure all created documentation follows the project's AI-optimized structure.
